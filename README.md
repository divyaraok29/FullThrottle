# FullThrottle

This is a Django application which produces data from database in JSON format as specified in the Test JSON.json file

To run this application in a windows machine follow the below steps:

<b>Prerequisites</b> <br>
<ul><li>Python 3.6.0 (Check your python version using <b><i>python --version</i></b>)</li></ul>

<b>Preparing Environment for Django Project</b>
<ol>
  <li> Create a virtual environment for this project (optional) </li>
  <ol>
    <li> Run <b><i>pip install virtualenvwrapper-win</i></b> command to install</li>
    <li> Now, create a virutal environment, <b><i>mkvirtualenv my_django_environment</i></b></li>
    <li> Enter the virtual environment created above by running <b><i>workon my_django_environment</i></b>. All the work from now on will be done in this virtual environment itself</li>
  </ol>
  <li> Now, install django with version 3.1 in the django environment created above, <b><i>pip install django~=3.1</i></b></li>
</ol>

<b>Once the above steps are completed, you have a Django environment ready!!</b>

<b>Running the application</b>
<ul>
  <li> First, clone the repository in your local machine</li>
  <li> Then, go to the directory, <b><i>useractivity</i></b> in your console</li>
  <li> Start the server by executing <b><i>python manage.py runserver</i></b></li>
</ul>

<b> Hola! The application is ready and running on your local server!! </b>
